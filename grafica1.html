Grafica la función sin(x) - csc(x) + 1 en un intervalo

Este código genera y grafica la función sin(x) - csc(x) + 1 en un intervalo específico, mostrando dónde cruza el eje x.

knitr::opts_chunk$set(echo = TRUE)

# Limpiar el entorno de trabajo
rm(list = ls())

# Crear un vector de valores desde 2 hasta 2.8 con incrementos de 0.1
x <- seq(2, 2.8, by = 0.1)

# Definir la función grafica 

grafica <- function(x, inicia, termina) {
  
  # Definir la función cosecante que no está incluida en R por defecto
  
  csc <- function(x) {
    return(1/sin(x))
  }
  
  # Crear un vector para almacenar los resultados
  
  c <- 1
  F <- numeric(length(seq(inicia, termina, by = 0.1)))
  
  # Calcular valores para cada punto en el rango
  
  for (val in seq(inicia, termina, by = 0.1)) {
    F[c] <- sin(val) - csc(val) + 1
    c <- c + 1
  }
  
  return(F)
}

# Calcular y mostrar los valores de y
y <- grafica(x, 2, 2.8)
print(y)

# Crear la gráfica

plot(x, y, type = "l", col = "magenta", 
     main = "Gráfica de sin(x) - csc(x) + 1",
     xlab = "x", ylab = "f(x)")

# Añadir grid
grid()

# Añadir línea horizontal en y = 0

abline(h = 0, col = "black")

# Añadir líneas verticales en los puntos de interés

abline(v = 2.39, col = "blue")
abline(v = 2.52, col = "blue")

# Añadir un punto en el punto de interés (aproximación de la raíz)
points(2.472, 0, pch = 16, col = "red")

# Leyenda

legend("topright", legend = c("f(x) = sin(x) - csc(x) + 1", "Raíz aproximada"),
       col = c("magenta", "red"), lty = c(1, NA), pch = c(NA, 16))

Explicaciones adicionales

  1. Para marcar el punto específico que representa la aproximación de la raíz (donde la función cruza el eje x), he usado la función points de R.

  2. la función legend para añadir una leyenda que explique los elementos del gráfico, lo cual es una buena práctica para mejorar la legibilidad.

Este código produce una visualización de la función sin(x) - csc(x) + 1 e identifica el intervalo donde la función cruza el eje x (donde la función tiene valor cero), marcando los límites con líneas verticales azules y el punto aproximado de cruce con un punto rojo.
